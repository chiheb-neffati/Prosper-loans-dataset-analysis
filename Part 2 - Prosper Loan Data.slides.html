Prosper loans data analysis and visualisation¶

Investigation overview¶

In this investigation, I wanted to see how can some of the variables of this dataset affect other variables and how can they be used to predict things like the lender's yield or the lender's estimated return.

Dataset overview¶

The Prosper loans data is a dataset provided from Prosper Marketplace Inc, a company specializing in loans at low interest rates to the borrowers.
This network represents loans between members of the peer-to-peer lending network. The online marketplace allows borrowers to post listings requesting loans. Lenders bid on these listings, which become loans if the amount requested is fully funded by the lenders. The network is directed from lender to borrower. Each edge also contains the time at which the loan originated, the amount requested, the loan status, the credit grade or rating of the borrower, the lender rate and the borrower rate, etc...(source)
The dataset is comprised of 81 variables and contains 113937 entries, you can learn more about what each columns signifies by clicking this link

The Number of loans for each term length:¶

In this dataset, we have three unique values where each value represents a term length in months, these values are 12, 36 and 60.
The goal of this visualisation is to see which term length is the most popular in this platform.

The most common income range¶

The goal of this representation is to learn more about the income demographic of the prosper loans platform where each income range represents the following values(in dollars):

  • Not employed -> NotEmployed
  • 1-24,999 -> Low
  • 25,000-49,999 -> LowerMiddle
  • 50,000-74,999 -> Middle
  • 75,000-99,999 -> UpperMiddle
  • 100,000+ -> High
